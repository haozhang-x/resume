解决SyntaxError-Non-ASCII character ‘\xe5′ in file

发表于 2017-03-17 |

问题：

编写Python时，尝试使用中文字符出现：

SyntaxError: Non-ASCII character ‘\xe5′ in file

原因是Python的默认编码文件是用的ASCII码，文件中使用了中文等非英语字符。

解决方式：

在Python源文件的最开始一行，加入一句：

# coding=UTF-8（等号换为”:“也可以）

或者

# -- coding:UTF-8 --
